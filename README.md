# 🧠 Reality Engine — Autonomous Decision System

> **"Stop building chatbots. Start building decision engines."**

Reality Engine is a high-fidelity autonomous system designed to solve the most critical bottleneck in modern enterprise: **Decision Velocity.** 

It doesn't just "chat" about problems—it observes live system states, detects constraints, simulates multiple possible futures, and executes the optimal strategic action in real-time.

---

## ⚡ The Pipeline (The 6-Agent Architecture)

The system operates through a deterministic chain of specialized agents, moving from raw data to measurable impact in seconds:

1.  **Observer Agent**: Monitors live inventory, vendor lead times, and order fulfillment SLAs.
2.  **Simulator Agent**: Generates exactly 3 distinct strategic "Futures" (e.g., Rush orders vs. Vendor switching).
3.  **Evaluator Agent**: Scores every future based on ₹ efficiency, risk, and delivery speed.
4.  **Decision Agent**: Selects the optimal path with structured, logical reasoning.
5.  **Executor Agent**: Applies the chosen action to the system state.
6.  **Auditor Agent**: Generates a tamper-proof execution trace for every autonomous decision.

---

## 🎨 Enterprise-Grade UI

Designed with the aesthetic of **Stripe** and **Linear**, the dashboard provides a high-signal, low-noise environment for strategic oversight:

*   **Live Environment State**: Real-time tracking of vendors and order flows.
*   **Decision Timeline**: A step-by-step visual "thinking process" of the AI engine.
*   **Strategic Futures**: Side-by-side comparison of generated options with "Why not chosen" logic.
*   **Outcome Comparison**: Horizontal visual analytics showing profit gains vs. baseline losses.
*   **Optimal Decision Focus**: High-visibility highlighting of the selected strategy and its ₹ impact.

## 🚀 Quick Start

### 1. Backend (Python 3.10+)
```bash
cd backend
pip install -r requirements.txt
python -m backend.main
```

### 2. Frontend (Node 18+)
```bash
cd frontend
npm install
npm run dev
```
